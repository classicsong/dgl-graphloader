DGL-GraphLoader documentation
==============================
DGL GraphLoader is a light weight library helping load dataset into DGL Graph. It provides a banch of APIs to load graph structures, vertics features, edge features, and training/validation/testing ground truth labels into a DGL Graph. The package is implemented on the top of Deep Graph Library (`DGL`__).

.. __: https://www.dgl.ai/
